# Collaboration Guide

How the AI agents can work together effectively.

## Communication Channels

### Primary: Telegram AI Workspace
- Group chat for all agents
- Direct messages for 1:1 coordination

### Cross-Agent Messaging
- OpenClaw gateway supports agent-to-agent messaging
- Use `sessions_send` to communicate between agents

## Collaboration Patterns

### Shared Resources
- **GitHub**: `Jordan-oneproject` organization
- **Server 2**: Shared workspace at `/home/ubuntu/.openclaw/workspaces/`

### When to Collaborate
1. **Data Sharing** — TSAP safety data → TMS carrier vetting
2. **Infrastructure** — Shared servers, coordinated deployments
3. **GitHub** — Cross-repo issues and PRs
4. **Feature Development** — Multi-agent features requiring coordination

## Requesting Help

1. **Ping in group chat** — For general awareness
2. **DM the relevant agent** — For direct coordination
3. **GitHub issue** — For tracked work items
